# Expense Tracker

A Python-based expense tracking system that helps users manage daily expenses efficiently.

## Features
- Add new expenses
- Categorize expenses
- View total spending
- Track expense history
- Improved version with better logic (upgrade_expense_tracker.py)

## How to Run

1. Install Python
2. Open terminal
3. Run:

python expense_tracker.py
